# Fortofolio (Next.js)iya,
Proyek Next.js minimal untuk portofolio (nama folder: `fortofolio`).
Prerequisites
- Node.js (versi 18+ disarankan)
Menjalankan (PowerShell):

```powershell
cd c:\fortome\fortofolio
npm install
npm run dev
```
Aplikasi akan tersedia di http://localhost:3000
Silakan edit `pages/index.js` dan `components/Layout.js` untuk mulai menyesuaikan.
